1. ТРЕБОВАНИЯ К ОБОРУДОВАНИЮ И ПРОГРАММНОМУ ОБЕСПЕЧЕНИЮ

Программные средства

Система управления данными "IntegralSystem" тестировалась на следующих операционных системах:
Windows® XP Professional or Home Edition + Service Packs 1 + Service Packs 2 + Service Packs 3 (рекомендуется)
Windows® Server 2003
Windows® Vista
Windows® Server 2008
Windows® 7
Windows® 8
Windows® 8.1
Windows® 10

Для работы системы "IntegralSystem" требуется установка следующего программного обеспечения:
PostgreSQL 9.2.0
Microsoft Visual C++ 2010 Redistributable
Microsoft Visual C++ 2015 Redistributable
Microsoft Internet Explorer 8 и выше

Аппаратные средства

Процессор семейств Intel® (Pentium® / Celeron® / Xeon™ / i-3 / i-5 / i-7), AMD (Athlon™ / Duron™) тактовая частота которых составляет не менее 200 МГц.
Минимальные требования к оперативной памяти зависят от используемой операционной системы: для Windows XP/2000/NT 4.0 - 64 Мб; Windows Me/98 - 32 Мб. При работе в многопроцессорных системах требуется дополнительно 16 Мб оперативной памяти для каждого дополнительного процессора.
Видеокарта и монитор с разрешением не менее 800х600 точек, глубина цвета не менее 16bit. Рекомендуемые параметры: разрешение 1280х1024 и более, глубина 16bit и более, мелкий шрифт, стандартная или классическая цветовая схема.
Свободное место на жестком диске не менее 100 Mb (900Mb при использовании HDD в качестве источника данных).
Клавиатура.
Манипулятор мышь.
Для сетевого варианта наличие локальной вычислительной сети. Установленный протокол TCP/IP.
