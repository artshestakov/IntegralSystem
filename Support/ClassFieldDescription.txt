1) Type - Тип поля
2) Size - размер значения поля
3) Upper и Lower - отвечают за регистр введенного текста: верхний и нижний соответственно;
4) DefaultValue - значение по умолчанию для поля таблицы;
5) DefaultValueWidget - значение по умолчанию для поля редактирования информации (наследник от ISFieldEditBase);
6) LabelName - текст отображаемый напротив поля редактирования информации в форме объекта;
7) LocalListName - текст отображаемый в заголовке таблицы;
8) NotNull - поле обязательно для заполнения;
9) ReadOnly - поле редактирования информации доступно только для чтения;
10) HideFromObject - отвечает за отображение поля редактирования информации в форме объекта (Поле создается, но не отобажается. В данном случае применяется setVisible для виджета поля);
11) HideFromList - поле не включается в запрос на выборку данных;
12) NotSearch - поиск по полю не будет осуществляться;
13) Hint - всплывающая подсказка с текстом указанным в значении этого параметра;
14) PlaceholderText - полу-прозрачный текст который будет отображаться если поле редактирования информации пустое (значение не введено/отсутствует);
15) ResizeColumn - автоподгон ширины поля в таблицы в соответствии с его содержимым;
16) ControlWidget - намиенование класса который нужно использовать в качестве поля редактирования информации;
17) IsSystem - поле является системным;
18) QueryText - текст запроса для полей с типом QUERY;
19) Sequence - на данном поле работает перечисление
